
This script is used to manage users passwords and generate new ones
NOTE:
before starting the application create a text file PasswordManager.txt to save and make changes in the file. Adjust $passwordManager and $readMe variables path before starting the application
 
-Create user: creates new account and adds new generated or user create password
-GeneratePassword: generates a new password.
-AddPassword: adds new password for an app. You can not add password if an app name already exists
-GetPassword: shows the password of the corresponding application
-DeletePassword: deletes app's password
-DeleteUser: deletes users and all of the associated passwords
-GetInfo: prints user instructions
